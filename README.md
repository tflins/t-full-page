# t-full-page

造轮子之 TypeScript 编写 的 full-page 插件

使用 Rollup 构建

特性：

- 快速的构建全屏滚动页面
- 开箱即用，不依赖任何框架或库
- 适配 PC、移动端
- 文件体积极小，Gzip 压缩后仅 1kb 大小

## 起步

### 安装依赖

```shell
npm i
```

### 开发

```shell
npm run serve
```

### 编译

```shell
npm run build
```
